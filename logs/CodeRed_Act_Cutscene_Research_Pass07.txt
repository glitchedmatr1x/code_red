Code RED / Faction War research note — ACT + cutscene scan
Date: 2026-04-29

Scope checked:
- game 1.zip extracted RPFs: act.rpf, gringores.rpf, naturalmotion.rpf, psocache.rpf where reachable
- content.rpf SCXML cutscene UI file
- content.rpf manifest for related gringo/robbery/lasso/car/shop scripts
- existing content AI TR exports for shopkeeper, hostile, vehicle, lasso, horse/mount behavior hooks

Main conclusion:
- The strongest next pass should not depend on cutscene UI. The useful gameplay hooks are ACT animation/task resources, gringo dictionaries/scripts, content AI TR, and content ambient events.
- act.rpf contains the exact named WAT action resources we need for lasso/hogtie/shop robbery/horse/car/vehicle behavior.
- gringores.rpf has a location-specific thieveslanding.wgd dictionary that references shopkeeper brains, bartender brain, storeowner companions, customer/dealer/counter activities, horse items, and store/shop talk entries.
- content.rpf contains the gringo scripts that probably bind those actions: lassogringo.wsc, pickuphogtiedgringo.wsc, hostage_kneel.wsc, shopgringo.wsc, car_gringo.wsc, playercar.wsc, vehicle_generator.wsc, gringo_actorhasweapon.wsc, carcrank_gringo.wsc, and Gen_Vehicle_Brain.

Cutscene findings:
- content/0x1B2D32DB/cutscenes.sc.xml is mostly UI state: pause during cutscenes, subtitle queue clearing, movie prompts, binoculars UI, controller disconnect, close request, quit movie, zoom movie.
- It is useful for UI behavior only, not for Faction War world spawning or robbery logic.
- I would not patch cutscenes.sc.xml for the next faction-war pass.

act.rpf overview:
- Entries: 1,035
- Files: 1,026
- Primary extension: .wat, 1,022 files
- WAT resources are RSC resource streams. Many are RSC05/RSC85 with Zstandard payloads.
- These are not plain text configs, but their resource names and decoded strings identify valid action/animation/task vocabulary.
