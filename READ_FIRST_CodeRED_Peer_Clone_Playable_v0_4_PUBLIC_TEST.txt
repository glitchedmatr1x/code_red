CODE RED PEER CLONE PLAYABLE v0.4 - PUBLIC TEST
=================================================

This is the next public test pass.

It is NOT the Red Dead game hook yet.
It does NOT go inside the game folder.
It does NOT edit RPF archives.
It does NOT restore official multiplayer yet.

What it does:
- Player A can host and play in a shared top-down arena.
- Player B can join by LAN/VPN IP.
- Both players can move around and see each other live.
- A solo bot can join so the host can test without another person.
- It writes bridge files for the next in-game clone actor bridge pass.

New bridge output:
- bridge/local_player_state.json
- bridge/remote_players_state.json
- bridge/bridge_status.json

These files are the simple target for the future game hook:
- read the local player state
- read remote player clone state
- spawn/move a clone actor inside the game

Requirements:
- Windows 10 or newer
- Python 3.10 or newer
- Same folder on both PCs
- LAN or VPN connection between both players

Controls:
- WASD or Arrow keys: move
- Shift: boost
- Space: pulse
- B: force bridge file update
- Q: quit
