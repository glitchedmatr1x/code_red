# Pass 1 Result

Added a Code RED Vehicle Script Lab drop-in.

The tool supports:

- status check
- target map generation
- binary/text script scanning
- folder scanning
- RPF script extraction/scan through Code RED's existing `python_workbench.py` backend
- playercar vs wagon thief comparison
- ASI scaffold creation

The tool is intentionally conservative. It turns WSC files into research maps and moves new behavior experiments to an ASI/ScriptHook bridge.
