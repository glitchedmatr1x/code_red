# Public Repo vs Private Dev Kit

Code RED can be both a public project and a private working dev kit, as long as the boundary stays clear.

## Public Repo

The public repo contains source code, documentation, schemas, examples, manifests, and clean-room research summaries.

## Private Dev Kit

The private dev kit can contain game files, extracted scripts, patch experiments, logs, drop-ins, and generated outputs. Keep it outside the repository.

## Rule

If a file came from the game, from extraction, from a local run, or from a private experiment, keep it private unless it has been reduced to a public-safe summary or original tool source.
