# Safety and Legal Notes

Code RED does not provide game files, cracked files, bypasses, or Rockstar-owned assets.

Users must supply their own legally owned game files outside the public repository. Public contributions should contain original source code, documentation, schemas, examples, manifests, and clean-room research summaries only.

Do not publish:

- Full RPF archives
- Extracted retail WSC/SCO/CSC/XSC scripts
- Models, textures, audio, video, string tables, or other game assets
- Third-party mods without permission
- Compiled binaries in the default branch
- Private logs or machine-specific output

When in doubt, keep the file private and document how users can generate it locally.
