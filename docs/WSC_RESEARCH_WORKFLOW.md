# WSC Research Workflow

Code RED may document script research, but the public repository should not include extracted retail script files.

## Public-Safe Material

- Original tooling
- Clean-room pseudocode
- High-level behavior notes
- Enum observations
- Path and string references when needed for context
- Validation reports that do not include raw script payloads

## Private Dev Kit Material

- Extracted WSC/SCO/CSC/XSC files
- Decompiler output
- Raw script payloads
- Drop-in binary patches
- Local test logs

Keep private material outside the public repo.
