# CodeRED NPC Roster Packs

Generated by `tools/codered_actor_enum_tool.py roster-pack` from `data/codered/actor_enum_map.csv`.

| File | Intended use |
|---|---|
| `npc_roster_law_gang_companion.txt` | Default testing roster for bodyguards, lawmen, gangs, and player-like actors. |
| `npc_roster_lawmen.txt` | Focused US Army/law roster for companion/bodyguard tests. |
| `npc_roster_gangs.txt` | Criminal/gang actors for hostility and target tests. |
| `npc_roster_mounts_vehicles.txt` | Rideable animals and vehicle actor enums. |
| `npc_roster_all_spawnable.txt` | Broad actor-only research roster; can be large. |

Point `CodeRED_AI_Menu.ini` `[paths] roster=` at one of these files, or copy entries into `data/codered/npc_roster.txt`.
