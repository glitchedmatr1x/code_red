Code Red Arcade replaceable sounds

Replace any of these files with your own .wav, .ogg, or .mp3.
The game loads every file whose name starts with one of these event names:

fire_left
fire_right
hit
break
explosion
pickup
boost
jump
land
engine

Examples:
  fire_left_heavy.wav
  explosion_big.ogg
  boost_loop.mp3

The shipped WAVs are tiny placeholder tones so the folder structure works immediately.
