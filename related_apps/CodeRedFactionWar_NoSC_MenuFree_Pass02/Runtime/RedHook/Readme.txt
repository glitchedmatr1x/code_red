[Red Hook v0.8]

How to install ?

• Simply drag and drop every files from this archive inside your Red Dead Redemption folder.
• Install any RED plugins of your choice (My Native Trainer here: https://www.nexusmods.com/reddeadredemption/mods/395)
• Launch the game
• Enjoy :)

F.A.Q:

What is the purpose of RedHook compared to other ScriptHook around ?
• RedHook features a clean console UI that allow you to reload/unload scripts on the fly, without having to restart your game. As well as featuring a trainer with a clean and modern UI.

What is a RED plugin ?
• This is the exact same as other mod using ASI plugins, the name change for the purpose of not clashing with other mods using ASI extension.

Known Issues:
• Mouse can freeze when F8 console is open in game, I suggest not using the mouse for now, and only type commands, or just pause the game for now and you will gain mouse control back. (I'm investigating the issue and will try to fix it for the next update)
