Code RED - Followers + Empty World Events Pass

Use this as a test patch, not as a permanent overwrite until it is verified in-game.

Install:
1. Back up your original content.rpf.
2. Copy the included content.rpf into your test game/content location.
3. Start the game.
4. Test wilderness areas far from roads, train lines, and horse trails.

Expected effect:
- More ambient events become eligible outside the empty world.
- Wilderness should be more likely to pull transport, escort, rescue, wagon, campfire, chase, and landmark-attack style events.
- Follower-like/approach events that were previously denied by the pathfinding deny set are now allowed to try.

Not guaranteed:
- This does not force permanent Dutch/Javier/Bill companions yet.
- It improves the event eligibility layer first. Permanent named followers probably need a deeper script/taskmachine/freemode hook.

Rollback:
- Restore your backed-up original content.rpf.
