CODE RED PEER CLONE PLAYABLE v0.3
=================================

This is the first playable public test.
It is NOT the Red Dead game hook yet.
It does NOT go inside the game folder.
It does NOT edit RPF archives.

What it does:
- Player A can host and play in a shared top-down arena.
- Player B can join by LAN/VPN IP.
- Both players can move around and see each other live.
- This proves the control + clone sync loop before we wire it into the game.

Requirements:
- Windows 10 or newer
- Python 3.10 or newer
- Same folder on both PCs
- LAN or VPN connection between both players

Controls:
- WASD or Arrow keys: move
- Shift: boost
- Space: pulse
- Q: quit

ONE-PC TEST FIRST:
Run:

Run_SelfTest.bat

Expected:

Selftest result: PASS
player_a saw player_b: True
player_b saw player_a: True

HOST / PLAYER A:
1. Run Run_Doctor.bat.
2. Give Player B your LAN/VPN IP, not 127.0.0.1.
3. Run Run_Host_And_Play_A.bat.
4. If Windows Firewall asks, allow Python/private network access.
5. Move with WASD. Keep the game window open.

CLIENT / PLAYER B:
1. Run Run_Join_Player_B.bat.
2. Type Player A's LAN/VPN IP.
3. Move with WASD.

SUCCESS:
- Both players see each other moving in the top-down Code RED arena.
- runtime/*.jsonl logs appear.

If this passes, the next phase is the real game bridge that reads the player position and moves a spawned clone actor inside the game.
