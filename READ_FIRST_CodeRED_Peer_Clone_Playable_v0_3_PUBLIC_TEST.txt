CODE RED PEER CLONE PLAYABLE v0.3 - PUBLIC TEST
=================================================

This is the first playable two-player Code RED connection test.

It is NOT the Red Dead game hook yet.
It does NOT go inside the game folder.
It does NOT edit RPF archives.
It does NOT restore official multiplayer yet.

What it does:
- Player A can host and play in a shared top-down arena.
- Player B can join by LAN/VPN IP.
- Both players can move around and see each other live.
- This proves the control + clone sync loop before the in-game clone actor bridge.

WHERE TO FIND THE FILES
-----------------------

public_tests/CodeRED_Peer_Clone_Playable_v0_3/

Also available as local zip artifact:

CodeRED_Peer_Clone_Playable_v0_3.zip

REQUIREMENTS
------------
- Windows 10 or newer
- Python 3.10 or newer
- Same folder on both PCs
- LAN or VPN connection between both players

Recommended VPN for remote testers:
- Radmin VPN
- ZeroTier
- Hamachi

WHERE TO PUT IT
---------------
Extract or copy the folder anywhere easy, for example:

D:\Games\CodeRED_Peer_Clone_Playable_v0_3\

Do not put it in the Red Dead Redemption game folder yet.

CONTROLS
--------
- WASD or Arrow keys: move
- Shift: boost
- Space: pulse
- Q: quit

ONE-PC TEST FIRST
-----------------
Run:

Run_SelfTest.bat

Expected:

Selftest result: PASS
player_a saw player_b: True
player_b saw player_a: True

TWO-PC PLAYABLE TEST
--------------------

HOST / PLAYER A:
1. Run:

   Run_Doctor.bat

2. Give Player B your LAN/VPN IP, not 127.0.0.1.
3. Run:

   Run_Host_And_Play_A.bat

4. If Windows Firewall asks, allow Python/private network access.
5. Move with WASD. Keep the window open.

CLIENT / PLAYER B:
1. Run:

   Run_Join_Player_B.bat

2. Type Player A's LAN/VPN IP.
3. Move with WASD.

SUCCESS LOOKS LIKE
------------------
- Player A sees Player B moving in the arena.
- Player B sees Player A moving in the arena.
- runtime/*.jsonl logs are created.

NEXT PHASE
----------
If this playable sync works, the next phase is the real game bridge that reads player position and moves a spawned clone actor inside the game.
