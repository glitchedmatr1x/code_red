Ninja Ripper 2 .NR Blender Importer - Clean Add-on Package

No compiling is required.

Recommended Blender version:
- Blender 2.93 LTS portable is the safest target.
- The add-on source says it was tested in Blender 2.79b, 2.83 LTS, 2.9, and 2.93.1.
- Newer Blender versions may need small API fixes.

Install:
1. Open Blender.
2. Edit > Preferences > Add-ons > Install.
3. Select this ZIP file.
4. Enable "Import Ninja Ripper 2".
5. Use File > Import > Ninja Ripper 2 World Space or Local Space.

Use:
1. First capture a game scene/model with Ninja Ripper 2, producing .nr files.
2. Import the .nr file into Blender with this add-on.
3. Use the imported meshes/textures as visual reference for remastering or matching archive assets.

This does not replace anything in-game. It imports captured Ninja Ripper 2 data into Blender.
