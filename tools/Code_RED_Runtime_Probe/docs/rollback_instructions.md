# Code RED Runtime Probe Rollback Instructions

To remove the runtime probe:

1. Delete `CodeRED_Runtime_Probe.asi` from beside `RDR.exe`.
2. Optionally delete `data\codered\runtime_probe.ini`.
3. Optionally archive or delete `data\codered\runtime_probe.log`.

No RPF, EXE, XML, WSC, or save files are changed by this probe.
